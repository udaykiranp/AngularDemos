Directive as an element attribute

All the angular directives are declared as element attributes
For Example : ng-click ng-controller 
          <html ng-app="app">
            <body ng-controller="myCtrl">
            ........
            Some html code here
            .......
            </body>
          </html>

Directive as a custom tag

Here we can decalre our own custom angular directive and insert them into html page as an element.
To write a custom directive we need to specify in the angular code as 
          angular.module('your module name').directive('directive name',function(){
            return{
                  attributes here ...
            }
          });
          Note : directive name should be 'camelCase' since javascript is case-sensitive

For Example : 
          angular.module('app').directive('studentDetailsCard',function(){
            return{
                  attributes here ...
            }
          });
At this stage we created a custom direcive named 'studentDetailsCard' with proper naming convention
We attach this directive to the module named 'app'
We can now specify this directive as an custom directive to an element. 
          <div student-details-card>....</div>

          Note: Here html is non case-sensitive so we need to specify the directive as above to get recognized by html.

Also we can specify 'x' or 'data' before custom directive and also we can use ('_') - underscore or (':') - colon instead of ('-') - hyphen or even mix them. This is called 'snake casing'
          <div data-student:details_card> ... </div>

In the attributes section of the directive we can specift 'template' attribute and sets its value to any html that you want to display when the directive has been included in html element.
          angular.module('app').directive('studentDetailsCard',function(){
            return{
              template: "Student Details here";
            }
          });
